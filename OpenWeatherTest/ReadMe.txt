Automation testing of the OpenWeatherMap API is done with the help of NUnit Framework with the support of RestSharp Http library 

Requirements to run the application:

Install the below packages once the code is cloned locally.
1)RestSharp - version used 106.15.0

Structure of the OpenWeatherTest project

Models folder 
Consists of all the clases which are the breakdown structure of the response JSON

Tests folder
Consists of all the areas which are interelated to the testing scenarios as mentioned in the manual test document.
